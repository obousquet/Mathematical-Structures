Rng

Name: Rng

Definition:

  • $(R, \cdot)$ is a semigroup,

Rings

Name: Rings

Definition:

A ring $R$ is a set equipped with two binary operations $(+, \cdot)$ such that

  • $(R, +)$ is an abelian group (i.e. addition is associative, commutative, has a zero and inverses)
  • $(R, \cdot)$ is a monoid (i.e. multiplication is associative and has a unit)
  • Multiplication is distributive with respect to addition
Description:

A ring is a set endowed with two binary operations, addition and multiplication. It is an abelian group for addition, a monoid for multiplication and multiplication distributes over addition.

Integral Domains

Name: Integral Domains

Definition:

An integral domain $R$ is a commutative ring where any non-zero element has the cancellation property: $$ \forall a\in R, ab=ac\implies b=c $$

Description:

Equivalent characterizations:

  • An integral domain is a nonzero Commutative Ring with no nonzero zero divisors.
  • An integral domain is a Commutative Ring in which the zero ideal ${0}$ is a prime ideal.
  • An integral domain is a nonzero Commutative Ring for which every nonzero element is cancellable under multiplication.
  • An integral domain is a Ring for which the set of nonzero elements is a commutative monoid under multiplication (because a monoid must be closed under multiplication).
  • An integral domain is a nonzero Commutative Ring in which for every nonzero element $r$, the function that maps each element $x$ of the ring to the product $xr$ is injective. Elements $r$ with this property are called regular, so it is equivalent to require that every nonzero element of the ring be regular.
  • An integral domain is a Ring that is isomorphic to a subring of a Field. (Given an integral domain, one can embed it in its field of fractions.)

Contained in: None

Contains: None

Integrally Closed Domain

Name: Integrally Closed Domain

Definition:

An integrally closed domain $R$ is an integral domain such that if $x$ is a root of a monic polynomial with coefficients in $R$, then $x\in R$.

Description:

An integrally closed domain $R$ is an Integral Domain whose integral closure in its field of fractions is $R$ itself. This means that if $x$ is an element of the field of fractions of $R$, i.e. a root of a monic polynomial with coefficients in $R$, then $x$ is itself an element of $R$.

GCD Domain

Name: GCD Domain

Definition:

A GCD domain is an integral domain with the property that any two elements have a greatest common divisor.

Description:

A GCD domain is an Integral Domain with any of the equivalent properties:

  • Any two elements have a greatest common divisor
  • There is a unique principal ideal containing the ideal generated by two elements
  • Any two elements have a least common multiple

Dedekind Domain

Name: Dedekind Domain

Definition:

A Dedekind domain is an integral domain in which every nonzero proper ideal factors into a product of prime ideals.

Description:

A Dedekind domain $R$ is either a Field or an Integral Domain satisfying any of the following equivalent conditions (these are equivalent for an Integral Domain):

  1. Every nonzero proper ideal factors into primes.
  2. $R$ is ?#noetherian and the localization at each maximal ideal is a ?#discrete_valuation_ring
  3. Every nonzero fractional ideal of $R$ is invertible
  4. $R$ is an Integrally Closed Domain, ?#noetherian with Krull dimension $1$.
  5. For any two ideals $I$ and $J$ in $R$, $I$ is contained in $J$ if and only if $J$ divides $I$ as ideals, i.e. there exists an ideal $H$ such that $I=JH$.

Unique Factorization Domain

Name: Unique Factorization Domain

Definition:

A unique factorization domain (UFD) is a ring $R$ in which every non-zero element $x$ which is not a unit can be written as a finite product of irreducible elements $$ x= p_1p_2\ldots p_n$$ with $n\ge 1$ and this representation is unique in the sense that if $x=q_1\ldots q_m$ with $m\ge 1$ then $m=n$ and there exists a bijection between the $p_i$ and the $q_i$ such that the image $q_j$ of $p_i$ is associated with $p_i$.

Euclidean Domain

Name: Euclidean Domain

Definition:

An Euclidean domain $R$ is an integral domain that can be endowed with a function $f:R \backslash \{ 0 \}\to \mathbb{N}$ such that for any $a,b\in R$ with $b\neq 0$, there exists $q,r\in R$ such that $$a=bq+r \mbox{ and } (r=0\mbox{ or } f(r)<f(b))$$

Field

Name: Field

Definition:

A field is a commutative ring where every non-zero element has a multiplicative inverse.

Description:

A field is a Commutative Ring where every element nonzero element is a unit, its nonzero elements thus form a multiplicative abelian group.

A commutative Division Ring is thus the same thing as a Field.
