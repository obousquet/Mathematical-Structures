Map of Categories

Name: Map of Categories

Definition:

A map between categories $C$ and $D$ is a pair $(o,m)$ of two maps, $o$ mapping objects of $C$ to objects of $D$ and $m$ mapping the morphisms of $C$ to morphisms of $D$.

Description:

This is not very interesting if we don't impose any constraints. But it is the base of our hierarchy of concepts.

Source and Target Preserving Map

Name: Source and Target Preserving Map

Definition:

A source and target preserving map between categories is a Map of Categories $(o,m)$ such that the source and targets of the morphisms are preserved: $$ f:A\to B \implies m(f):m(A)\to m(B) $$

Functor

Name: Functor

Definition:

A Functor $F$ between categories $C$ and $D$ is a Map of Categories such that

  • $F$ associated $f:A\to B$ with $F(f):F(A)\to F(B)$ (it is a Source and Target Preserving Map)
  • $F(g\circ f)=F(g)\circ F(f)$ (it preserves composition)
  • $F(id_{A}) = \text{id}_{F(A)}$ (it preserves identity morphisms)

Full Functor

Name: Full Functor

Definition:

A Functor $F:C\to D$ is called full when the induced function on Hom sets $$ F_{X,Y}: Hom_C(X,Y)\to Hom_D(X,Y) $$ is surjective.

Description:

A Functor is full when it is surjective on Hom sets.

Essentially Surjective Functor

Name: Essentially Surjective Functor

Definition:

Right Adjoint Functor

Name: Right Adjoint Functor

Definition:

A Functor $F:C\to D$ is a right adjoint if there exists a functor $G:D\to C$ and a natural isomorphism $Hom_D(F(X),Y) \cong Hom_C(X,G(Y))$ for all $X$ in $C$, $Y$ in $D$.

A Functor is right adjoint if it is part of an adjunction as the right adjoint.

Isomorphism of Categories

Name: Isomorphism of Categories

Definition:

A Functor $F:C\to D$ is an isomorphism of categories if there exists a functor $G:D\to C$ such that $F \circ G = id_D$ and $G \circ F = id_C$.

Description:

A Functor is an isomorphism if it has an inverse functor and both compositions are the identity.

Pseudomonic Functor

Name: Pseudomonic Functor

Definition:

A Functor $F:C\to D$ is pseudomonic if it is faithful (injective on hom-sets) and full on isomorphisms: for any objects $x, y$ in $C$, the map $F: \mathrm{Iso}_C(x, y) \to \mathrm{Iso}_D(Fx, Fy)$ is bijective.

Description:

A Functor is pseudomonic if it is faithful and full on isomorphisms.

Split Essentially Surjective Functor

Name: Split Essentially Surjective Functor

Definition:

A Functor $F:C\to D$ is split essentially surjective if for every object $Y$ in $D$, there exists an object $X$ in $C$ and an isomorphism $F(X) \cong Y$ in $D$ such that the isomorphism is split (i.e., there is a chosen inverse).

Description:

A Functor is split essentially surjective if every object in $D$ is isomorphic to the image of some object in $C$ with a chosen inverse.
