Monadic Functor

Name: Monadic Functor

Definition:

A Functor $F:C\to D$ is monadic if it is equivalent to the forgetful functor from the category of algebras for a monad on $D$ to $D$.

Description:

A Functor is monadic if it arises as the forgetful functor from the category of algebras for a monad.
